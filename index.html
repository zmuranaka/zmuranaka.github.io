About Me

I love to code and am proficient with the languages HTML, CSS, JavaScript, PHP, C, C++, C#, Java, Python, and SQL. Hypergeometric Calculator

Responsive web application that solves hypergeometric distributions, which describe the probability to get a number of desired successes out of a number of draws from a population without replacement. I built this mainly with JavaScript, which performs the calculations and writes them to the HTML.
