Custom Game Engine

Custom-built game engine that allows the user to create their own SPA (Single Page Application) with no coding experience. Built in an fully remote agile development team with a Django REST Framework back-end and an Angular and Bootstrap front-end. Hosted version uses Heroku and SQLite to demonstrate two games including original pixel art.
